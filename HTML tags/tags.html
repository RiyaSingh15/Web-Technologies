'h' tag is used to show heading. H tag has 6 types 'h1', 'h2', 'h3', 'h4', 'h5', 'h6'.

'h1' tag is used to show highest priority heading.

'h2' tag is used to show second highest priority heading.

'h3' tag is used to show next highest priority heading.

'h4' tag is used to show next highest priority heading.

'h5' tag is used to show next highest priority heading.
'h6' tag is used to show next highest priority heading.

'p' tag is used to create a paragraph. All the text entered under 'p' tag is considered as one paragraph irrespective of entering into next line. Which means 'p' tag omits extra spaces and newline.
Still if we want the text to be displayed in next line we use 'br' tag. Or if we want to display mutiple spaces as it is we use 'pre' tag. 

 'pre' tag displays all te spaces '            ' as it is.

'div' tag is used to create a division for the text it is more or less same as 'p' tag.
'sup' tag is used to show superscript
Similarly 'sub' tag is used to show subscript
'marquee' tag is used to display scrolling text.
